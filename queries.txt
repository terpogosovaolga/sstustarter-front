1. ГЛАВНАЯ СТРАНИЦА
- информация о сайте (название, описание, картинка)
- количество проектов (ProjectController)
- темы-родители (ThemeController)
- топ посещаемых проектов (6)
- ссылки на соцсети (ссылка, картинка)

2. КАТАЛОГ
- проекты с пагинацией 
- сортировка, фильтрация
- поиск 

3. ЛИЧНЫЙ КАБИНЕТ
- данные пользователя 
- количество созданных проектов 
- количество задействованных проектов 
- популярное направление (по задействованным и созданным проектам)

- Post редактирование аватара 
- Post редактирование данных (ФИО, структуры, текста)
- Post редактирование данных для входа (логина, пароля)

- созданные проекты: короткая инфа (картинка, тема, родительская тема, название, автор - фио должность). статус модерации (сообщение модерации)
- Post редактирование проекта, отправка на модерацию
- задействованные проекты: короткая инфа та же + наличие новых заявок 
- проекты, где пользователь желающий: короткая инфа + заявка пользователя 
- Post перевод проекта на следующий этап 
- Post досрочное закрытие проекта??

- заявки (инфа от пользователя, название проекта и ссылка на него, текст заявки, дата отправки, статус заявки)
- Post ответ на заявку (принять, отклонить) 
- Post прочитать заявку 


4. ПРЕЗЕНТАЦИЯ ПРОЕКТА 
- Post просмотр проекта
- основная инфа + Об авторе + Кого ищем + Цели + Фото + количество участников (набрано заявлено)
- Post отправка заявки

- все проекты (пагинация)
- фильтрация, сортировка, поиск 
- Post архивирование проектов 
- аналитика 

6. МОДЕРАТОР 
- все проекты (пагинация)
- фильтрация, сортировка, поиск 
- Post архивирование проектов 
- аналитика 
- Post редактура проекта (все тексты)

- Post ответ на модерацию (смена статуса, сообщение)
